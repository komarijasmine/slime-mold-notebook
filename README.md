Before starting the notebook, you should create a virtual environment.

For Windows:
# 1. Create the environment 
py -m venv geo_env

# 2. Activate it
.\geo_env\Scripts\activate

# 3. Install requirements / packages
py -m pip install -r requirements.txt

For MacOS/Linux:
# 1. Create the environment 
python3 -m venv geo_env

# 2. Activate it
source geo_env/bin/activate

# 3. Install requirements / packages
python3 -m pip install -r requirements.txt
